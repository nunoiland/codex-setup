# Contributing

## Korean | 한국어

기여해주셔서 감사합니다.
이 저장소는 Codex 설정 템플릿을 함께 다듬기 위한 오픈소스 프로젝트입니다.

### 기여 방법

1. 이슈를 확인하거나 새 제안을 남깁니다.
2. 작고 명확한 범위로 변경합니다.
3. 문서와 설정이 함께 맞는지 확인합니다.
4. 가능하면 변경 이유를 README, AGENTS, 스킬 문서 중 적절한 위치에 반영합니다.

### 권장 변경 예시

- README, NOTICE, PRD, PLANS 문서 개선
- `.agents/skills/`의 스킬 추가 또는 정리
- `.codex/agents/`의 에이전트 역할 개선
- Codex 설정값과 훅 구성 개선
- 오타, 표현, 구조 정리

### 요청 사항

- 큰 구조 변경은 먼저 이슈나 PR 설명에서 의도를 공유해주세요.
- 불필요하게 범위를 넓히기보다, 작고 읽기 쉬운 변경을 선호합니다.
- 검증을 실행하지 않았다면 실행했다고 적지 말아주세요.

## English

Thanks for contributing.
This repository is an open-source project for improving a reusable Codex setup template together.

### How to contribute

1. Review existing issues or open a new proposal.
2. Keep changes small and clear in scope.
3. Make sure docs and configuration stay aligned.
4. When useful, reflect the reason for the change in the README, AGENTS file, or relevant skill docs.

### Good contribution examples

- improving README, NOTICE, PRD, or PLANS docs
- adding or refining skills in `.agents/skills/`
- improving agent roles in `.codex/agents/`
- refining Codex config values or hook setup
- fixing typos, wording, or structure

### Requests

- For larger structural changes, explain the intent in an issue or PR description first.
- Prefer small, reviewable changes over broad unrelated edits.
- If validation was not run, do not claim that it was.
