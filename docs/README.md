# Docs Index

This repository is a worktree-first, PRD-first, validation-first, GitHub-first product template repo.

- GitHub is the canonical record for code, PRs, validation history, and handoff artifacts.
- Local handoff artifacts are the supported reporting path.
- Strategy, memory, and self-improvement are advanced capabilities, not the default operating path.
