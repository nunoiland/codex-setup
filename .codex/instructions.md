# Codex Instructions

- Work inside the current worktree only.
- Read the active PRD, `AGENTS.md`, `README.md`, and `docs/worktree-orchestration.md` before editing.
- For medium or large work, write or update a plan in `PLANS/` before making changes.
- Prefer existing patterns and focused diffs.
- Ask before dependencies, migrations, auth, payment, secrets or env handling, deployment, file deletion, destructive git actions, force push, or risky remote mutations.
- Default workflow is local-repo-first and GitHub-first. self-improvement flows are optional advanced paths only.
- End by reporting changed files, validation commands actually run, unsupported or removed config keys, and remaining risks.
